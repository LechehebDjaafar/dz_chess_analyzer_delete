```python
def calculate_win_rate(games_list):
    """
    حساب معدل الفوز من قائمة المباريات
    
    Args:
        games_list (list): قائمة المباريات
        
    Returns:
        float: معدل الفوز كنسبة مئوية
    """
    wins = sum(1 for game in games_list if game.result == 'win')
    return (wins / len(games_list)) * 100 if games_list else 0
```

### ❓ **المشاكل الشائعة وحلولها**

#### **🔍 خطأ "اللاعب غير موجود"**
```
الخطأ: "Player not found" أو "لا يمكن العثور على اللاعب"
```
**الحلول:**
- ✅ تأكد من صحة اسم المستخدم (حساس لحالة الأحرف)
- ✅ تأكد من أن الحساب عام وليس خاص
- ✅ جرب كتابة الاسم بأحرف صغيرة فقط
- ✅ تحقق من وجود الحساب على Chess.com مباشرة

#### **⏳ بطء في التحليل**
```
الخطأ: التحليل يستغرق وقتاً طويلاً أو يتوقف
```
**الحلول:**
- ⚡ قلل عدد المباريات المحللة (ابدأ بـ 20 مباراة)
- ⚡ تأكد من استقرار اتصال الإنترنت
- ⚡ أعد تشغيل التطبيق: `Ctrl+C` ثم `python app.py`
- ⚡ تحقق من ذاكرة النظام (يُفضل 8GB+)

#### **💾 خطأ في الجلسة**
```
الخطأ: "Session expired" أو فقدان البيانات
```
**الحلول:**
- 🔄 امسح بيانات المتصفح (Cache & Cookies)
- 🔄 أعد تشغيل الخادم وحديث الصفحة
- 🔄 تحقق من عدم انتهاء صلاحية الجلسة (4 ساعات)
- 🔄 تأكد من تفعيل الكوكيز في المتصفح

#### **🌐 مشاكل Chess.com API**
```
الخطأ: "API Error" أو "Connection failed"
```
**الحلول:**
- 🌍 تحقق من حالة Chess.com API
- 🕐 انتظر بضع دقائق وحاول مرة أخرى
- 🔄 أعد تشغيل التطبيق
- 🛡️ تحقق من إعدادات Firewall/Proxy

```python
            timeout=10
        )
        if response.status_code != 200:
            issues.append("Chess.com API غير متاح")
    except requests.RequestException:
        issues.append("لا يمكن الوصول لـ Chess.com API")
    
    # فحص مكتبات Python
    try:
        import flask, chess, pandas
        print("✅ جميع المكتبات متوفرة")
    except ImportError as e:
        issues.append(f"مكتبة مفقودة: {e}")
    
    return issues

if __name__ == "__main__":
    issues = system_health_check()
    if issues:
        print("🚨 مشاكل مكتشفة:")
        for issue in issues:

        sys.exit(1)
    else:
```

```bash

my-chess-env\Scripts\activate
```
